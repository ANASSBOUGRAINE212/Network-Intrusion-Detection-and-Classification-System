About Our Model

This system uses a Wide & Deep Neural Network trained on the CICIDS2017 dataset to detect and classify 15 different types of cyber attacks in real-time. The model analyzes 65 network traffic features including:

  • Temporal Features: Hour, minute, second (when the traffic occurred)
  • Flow Features: Flow duration, IAT (Inter-Arrival Time) statistics
  • Packet Statistics: Packet lengths, sizes, and distributions
  • Flow Metrics: Bytes/s, Packets/s, forward/backward ratios
  • TCP Flags: SYN, ACK, FIN, PSH, RST, URG flags
  • Connection Patterns: Active/Idle times, subflow statistics

The model achieves high accuracy in distinguishing between benign traffic and various attack patterns, making it suitable for real-time network intrusion detection systems.
